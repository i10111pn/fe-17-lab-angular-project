## Initial Setup

* check if `node` has 16.x.x version:

  `node -v`
* check if `npm` has 8.x.x version:

  `npm -v`

* check if `angular cli` has 13.x.x version:

  `ng --version`

* clone project: 

  `git clone https://github.com/i10111pn/fe-17-lab-angular-project.git`

* install dependencies:

  `npm install`

* run project:

  `ng serve -o`
